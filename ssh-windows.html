Generating SSH keys on Windows 10

1) Launching GitBash a) Open `C:\Program Files\Git\bin\bash.exe` b) Create the SSH directory in your root folder `mkdir .ssh` It will look something like this: `C:\Users\\.ssh` c) Generate the keys `ssh-keygen.exe` d) When prompted for a password, type in a password or leave blank and press enter to complete the process e) The output will look similar to the one shown below: Your identification has been saved in C:\Users\/.ssh/id_rsa. Your public key has been saved in C:\Users\/.ssh/id_rsa.pub.
